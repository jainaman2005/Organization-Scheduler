📅 Organization Scheduler API

Manage organizations, users, schedules, and queries with role-based access control.

✨ Features

  • 🔐 Authentication & Authorization with JWT
  • 🏢 Organization Management
  • 👥 User Roles (Admin 👑, Manager 🧑‍💼, Member 🙋)
  • 📅 Schedule Management (create, assign, update, delete)
  • ❓ Query Management (raise, respond, resolve)

🔗 API Endpoints

🔐 Authentication

  • POST /api/auth/login – Login a user
  • POST /api/auth/logout – Logout current user

🏢 Organizations

  • POST /api/orgs/register – Register a new organization (with Admin)
  • DELETE /api/orgs/:orgId – Delete an organization (Admin only)

👥 Users

  • GET /api/users/me – Get current user profile
  • PUT /api/users/me – Update profile
  • PUT /api/users/me/password – Change password
  • POST /api/users/add – Admin creates Manager/Member
  • GET /api/users/all-users – Admin lists all users
  • PUT /api/users/:id – Admin updates user
  • DELETE /api/users/:id – Admin deletes user
  • GET /api/users/members – Manager lists their members
  • POST /api/users/members – Manager creates Member

📅 Schedules

  • POST /api/users/schedules – Create schedule (Admin/Manager)
  • GET /api/users/schedules/managed – View schedules managed by user
  • GET /api/users/schedules/my – View schedules assigned to me
  • GET /api/users/schedules/:scheduleId – View specific schedule
  • PUT /api/users/schedules/:scheduleId – Update schedule (Admin/Manager)
  • PATCH /api/users/schedules/:scheduleId/status – Update schedule status (Member)
  • DELETE /api/users/schedules/:scheduleId – Delete schedule

❓ Queries

  • POST /api/users/schedules/:scheduleId/queries – Raise a query
  • GET /api/users/schedules/:scheduleId/queries – View all queries
  • POST /api/users/schedules/:scheduleId/queries/:queryId/responses – Add a response
  • GET /api/users/schedules/:scheduleId/queries/:queryId/responses – View responses
  • PATCH /api/users/schedules/:scheduleId/queries/:queryId/resolve – Resolve query
  • DELETE /api/users/schedules/:scheduleId/queries/:queryId – Delete query
